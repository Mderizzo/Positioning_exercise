# Posizionamento con CSS – Esercizio HTML + CSS

[![GitHub repo](https://img.shields.io/badge/GitHub-Repository-blue?logo=github)](https://github.com/tuo-username/tuo-repo)

Questo progetto è un esercizio pratico per comprendere il posizionamento degli elementi in una pagina web utilizzando le proprietà CSS `relative` e `absolute`.

La struttura HTML è composta da un contenitore principale di colore verde, al cui interno sono presenti due blocchi blu posizionati in modo assoluto. 
Ogni blocco blu contiene a sua volta due quadrati rossi, anch’essi posizionati con coordinate specifiche. 
L’obiettivo è esercitarsi con la gestione delle coordinate `top`, `left`, `right` e con l’uso combinato delle classi per ottenere layout precisi.
